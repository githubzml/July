# 第二章

参与人越多，出错的概率越大

墨菲定律

1. 任何事情都不会像它表面看起来那么简单
2. 所有任务完成的周期都会比你预计的时间要长
3. 任何事情如果有可能出错，那么必然会有极大的概率出错
4. 如果你预感可能会出错，那么它必然出错

幸运的是我们可以借此启示， 提前做一个比较周全的方案
